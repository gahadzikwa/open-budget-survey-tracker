Open Budget Survey
==================

# Install

## Dependancies
- __node & npm__
- __bower__ (for handling vendor libs) `npm install -g bower`

## Installation

  > npm install -g bower
  > git clone https://github.com/okfn/open-budget-survey-realtime.git
  > cd open-budget-survey-realtime
  > bower update
  > npm install
  > npm start
  
Open: http://localhost:3000/
